
# Late-Time Cosmic Acceleration from Coupled Higgs and Neutrino Mass Variation

## Manuscript

**Title:** Late-Time Cosmic Acceleration from Coupled Higgs and Neutrino Mass Variation

**Authors:** Malavika K, Soumya Chakrabarty, Arpita Mondal, and Arghya Choudhury

*The manuscript is currently in preparation. Further details and citation information will be added upon publication.*

## Overview

This repository contains the implementation of Markov Chain Monte Carlo (MCMC) analyses for the reconstruction of the Hubble expansion dynamics and the derivation of scalar field profiles, the dark energy equation of state, and neutrino coupling functions.

## Packages

The MCMC analyses are performed using the `emcee` Python package:

https://emcee.readthedocs.io/en/stable/

Symbolic derivations and analytical calculations are carried out using Mathematica.

## Installation

Installation instructions for `emcee` can be found in its official documentation:

https://emcee.readthedocs.io/en/stable/

A typical installation via `pip` is

```bash
pip install emcee
```
